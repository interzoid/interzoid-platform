





API Product Page



----------------







Individual Name Matching API Product Page: https://www.interzoid.com/apis/individual-name-matching







The API generates similarity keys for a given individual. A similarity key represents hundreds of possible permutations of an individual's full name (variations, misspellings, nicknames, typos, etc.). Then, matching by similarity key rather than the actual data itself, provides for dramatically higher match and search rates when comparing data.
